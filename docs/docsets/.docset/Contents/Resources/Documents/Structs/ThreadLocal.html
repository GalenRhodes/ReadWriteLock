ThreadLocal

@propertyWrapper
public struct ThreadLocal<T>

Thread Local Property Wrapper. A property marked with this wrapper will reserve storage for each thread so that the values gotten and set will only be seen by that thread.

NOTE: You should use caution with DispatchQueues. DispatchQueues reuse threads. This means that multiple items put onto a dispatch queue may be able see and manipulate data from another item on the queue.
